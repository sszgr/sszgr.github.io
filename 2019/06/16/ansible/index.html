ansible 笔记

发表于 2019-06-16

最近发现ansible挺好用的，简单做下笔记。

  • 默认通过SSH协议管理机器
  • 不用在远程机器安装软件
  • 快速上手
  • 可以使用非root用户

安装

RHEL and CentOS:
$ sudo yum install ansible
Fedora:
$ sudo dnf install ansible
Python-Pip:
$ sudo pip install ansible

配置

安装完成后的默认配置目录为/etc/ansible/
